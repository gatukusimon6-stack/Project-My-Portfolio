# Simon Gatuku – Software Developer Portfolio

A clean, responsive, multi-page personal portfolio website built using **HTML, CSS, and JavaScript**.

---

## 📌 Overview
This is my personal portfolio website showcasing my skills, projects, and background as a software developer.

## 🚀 Features
- Multi-page structure (Home, About, Projects, Contact)
- Responsive design for mobile and desktop
- Clean and modern UI
- Simple navigation system
- Project showcase section
- Skills section highlighting technical abilities

## 🛠️ Built With

- JavaScript (Vanilla JS)
